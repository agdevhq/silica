title: Backlinks
description: See which pages link to the current one.
---

When other pages link to the page you are reading, Silica lists them in a **Backlinks** section at the bottom. This makes it easy to discover related notes, just like in Obsidian.

This page is linked from the [[index|home page]], so you should see that backlink below.

Both kinds of internal links count toward backlinks:

- wikilinks — `[[other-page]]`
- standard Markdown links — `[text](/other-page)`

You do not configure anything; the connections are built from your links automatically whenever the site rebuilds.
