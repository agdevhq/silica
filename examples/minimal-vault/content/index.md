Silica turns a folder of Markdown into a polished, authenticated knowledge site.

Open [[notes/auth|Auth notes]], [[notes/search|Search notes]], [[notes/frontmatter|Frontmatter notes]], or [[notes/callouts|Callout examples]] to explore links and backlinks.

> [!note] Obsidian syntax
> Wikilinks, callouts, highlights like ==this text==, and assets such as ![[images/sample.svg]] are supported.

```ts
const framework = "Silica";
console.log(`${framework} renders markdown at request time and caches it.`);
```

Inline math works too: $E = mc^2$.
