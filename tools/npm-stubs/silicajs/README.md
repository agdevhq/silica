# silicajs

Reserved umbrella package for the upcoming **Silica** framework — a Next.js framework for publishing Obsidian-flavored markdown vaults with built-in Google OAuth, server-side search, and a Mintlify-like developer experience.

This `0.0.0` release is a placeholder reserving the name. The first real release will ship as `@silicajs/*` scoped packages.
